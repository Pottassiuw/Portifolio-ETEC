Conceitos Teóricos

FlatList: Um componente do React Native usado para exibir uma lista rolável de itens eficientemente. Gerenciamento de Estado Global: O conceito de manter o estado compartilhado em toda a aplicação, geralmente com o Context API ou uma biblioteca de gerenciamento de estado como Redux. Navegação entre Telas: O uso do React Navigation para criar transições entre diferentes telas do aplicativo. Filtragem e Pesquisa: A capacidade de filtrar e pesquisar dados com base em critérios específicos, como gênero ou ano. Design Responsivo: A criação de layouts que se ajustam automaticamente a diferentes tamanhos de tela.

Conceitos Práticos

Criação de um conjunto de dados de filmes. Uso do componente FlatList para exibir a lista de filmes de forma eficiente. Implementação de um mecanismo de filtro que permite ao usuário selecionar critérios de filtragem, como gênero ou ano. Utilização de um gerenciamento de estado global (por exemplo, com o Context API) para compartilhar o estado do filtro entre diferentes partes do aplicativo. Uso do React Navigation para criar uma experiência de navegação fluída entre a lista de filmes e a tela de detalhes de cada filme. Design responsivo para garantir que o aplicativo funcione bem em diferentes dispositivos móveis.
